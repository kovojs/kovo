# create-kovo

Starter generator for Kovo apps. It writes the application skeleton, package
scripts, local agent docs, and the first runnable app.

```sh
pnpm dlx create-kovo my-app
cd my-app
pnpm install
pnpm run check
```

```sh
pnpm create kovo my-app --template app
```
