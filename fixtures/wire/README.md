# Wire Fixtures

These are the Phase 0 hand-authored wire-format golden fixtures from the archived v1 roadmap.
They are contracts for the v1 transports: document load, no-JS form posts, enhanced mutation fetches, validation fragments, and `<kovo-defer>` streaming.

The SSE chunk fixture is intentionally not present. SSE and `<kovo-live>` are v2 backlog items.
